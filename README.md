# 🌱 Rayeva AI – Sustainable AI Backend System

## Overview
AI-powered backend system built using PHP & MySQL for sustainable commerce automation.

## Modules Implemented

### 1. AI Category Generator
- Generates primary category
- Sub category
- SEO tags
- Sustainability filters
- Stores structured JSON in MySQL

### 2. AI B2B Proposal Generator
- Suggests sustainable product mix
- Allocates budget
- Provides cost breakdown
- Includes impact summary
- Budget validation logic implemented
